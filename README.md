# EmbedKit_Aniket

Embedded C utility library implementation of a fixed-size circular buffer (Ring Buffer) for uint8_t data with a capacity of 8 bytes.

## Name
Aniket Ajit Wagavade

## Module Description

### Ring Buffer (ringbuf.c)
Implementation of a circular buffer (FIFO queue) supporting:
- Buffer initialization
- Byte write operation
- Byte read operation
- Buffer full detection
- Buffer empty detection
- Data count tracking
- Wrap-around handling
